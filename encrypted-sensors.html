CeLAN 1076 DRS Recessed Sensor Mahogany (Brown)

Ce-1076DRS

 

The CeLAN enabled GE 1076 Series Steel Door contacts are designed specifically for use in the steel doors commonly found in commercial building applications. The unique housing design features a rugged uni-body construction with flexible ribbed sides for quick, secure installation without gluing. The magnet housing isolates the magnet from the surrounding steel for maximum gap distances, both make and break. Both contact and magnet plastic housings are constructed of one piece of thick-walled ABS plastic for maximum strength and durability.

CeLAN WF 2-Zone Input Module

Ce-WF2ZI

 

The CINCH Systems CeLAN WF 2-Zone Input Module expands the versatility of Protector and Protector II intrusion control panel by providing a upgrade path for existing installed systems that use the Wells Fargo SM-3 contact switch. The CeLAN WF 2-Zone input module is designed to fit inside the SM-3 contact switch replacing the existing module inside the SM-3 contact. This module adds two dedicated (Contact & Cover Tamper) hardwire input zones to Protector and Protector II systems.  On the Protector and Protector II systems, multiple input modules can be connected  to a single panel for up to 256 input points.

The module monitors all zones and alerts the panel if there is an open/short circuit, if the contact is tampered or removed from the mounting.
