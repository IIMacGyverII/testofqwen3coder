CeLAN Printer Module

CE-P

 

The CeLAN Printer Module allows you to connect a parallel printer to any panel or receiver. The module allows you to print (or log) events as they occur. The module also includes a zone input that can be used as a tamper or any other system input.

  • CeLAN AES encrypted technology for security, easy setup and enhanced performance.
  • Single 25-pin “Centronics” Parallel Connector.
  • One (1) fully supervised input zone.
  • Five (5) state zones:
  • Open Circuit
  • Supervised Circuit (3.0k EOL)
  • Short Circuit
  • Ground Fault High
  • Ground Fault Low
  • On-board LED power and CeLAN bus status indicators.
  • Includes Accessory Hardware kit.
  • CINCH Stick Field Upgradeable firmware.

TCP Module

CE-TCP

 

A dual-application device that allows:

  • Stand-Alone Mode 1: Connect up to five separate supervised inputs for remote reporting, and remotely command one on-board relay.
  • Protector and Protector II CeLAN Mode: Communicate status remotely, interact with the Protector and Protector II System remotely (when allowed), and extend the CeLAN BUS via TCP/IP (when allowed).

When used in the stand alone mode the TCP/IP Module uses AES encryption to allow for secure communication between the module and a remote monitoring station. Status of up to 5 supervised inputs and remote operation of one on-board relay.

When used in Protector and Protector II CeLAN mode, the TCP/IP Module uses AES encryption for secure communications between Protector and Protector II System and TCP/IP Module, and:

  • Remote monitoring stations
  • Remote TCP/IP Modules
  • Remote SMS (Security Management Systems)

TCP/IP module comes with five 3.0K ohm EOL resistors and mounting hardware (screws and anchors).

Power for the CeLAN TCP/IP Module:

  • Protector and Protector II 12 VDC CeLAN power terminals.
  • CeLAN Remote Power Supply Module, or
  • Separate 12 VDC-power supply capable of delivering up to 250 mA
  • CeLAN AES encrypted digital data bus technology provides secure, fast and reliable data transfer between the Protector and Protector II Panel and CeLAN modules.

 

Features:

  • 128-bit AES Encrypted CeLAN communications.
  • 256-bit AES Encrypted TCP/IP communications.
  • Five (5) fully supervised input zones.
  • Five (5) state zones:
  • Open Circuit
  • Supervised Circuit (3k EOL)
  • Short Circuit
  • Ground Fault High
  • Ground Fault Low
  • One (1) Relay output
  • Guaranteed Delivery 32k on-board message buffer.
  • On-board LED power and CeLAN bus status indicators.
  • CeLAN AES encrypted technology for security, easy setup and enhanced performance.
  • Can be mounted in all compatible cabinet enclosures, sold separately.

Translator Module

Ce-T422

 

The CINCH Systems CeLAN RS422 Translator  Module expands the versatility of Rampart System by adding the capability to use 3rd party interface products to also control the operation of vehicle barriers, gates or other similar devices.

  • Connect both touch screen controls (Ce-TS-VBS) along with 3rd party controls (Proface, Indusoft, Wonderware, etc) to operate the same or different gates.
  • The Ce-T422 acts as the translator interface and firewall between the Rampart system and the 3rd party interface product.  Advanced design allows direct commands to either the Rampart panel or to 3rd party interface products.
  • Provides a consistent interface for reliable operation of the system that the integrator can use without requiring custom firmware to be written for the module for each project. The Ce-T422 Modbus interface allows the integrator to customize the 3rd party interface products to meet the needs of the customer.
  • For added security and reliability the Ce-T422 has dual CeLAN inputs so it can maintain a redundant, secure communications path to the rampart panel.  In redundant CeLAN mode the Rampart panel utilizes both CeLANs when sending messages to the module adding to the security provided by the AES encryption.  In the event of a single CeLAN failure the Rampart panel can still communicate with the module using the remaining CeLAN.

The Rampart System can monitor and control up to 20 vehicle barrier, gates or similar systems per panel and allows for up to 100 CeLAN devices to be connected to its dual CeLANs.

To simplify installation and troubleshooting there are 4 LEDs on Board:

- 1 - Green uC (microcontroller) operating normal LED.

- 2 - Red CeLAN communications LED.

- 1-  (Proface) 3rd party [RS422 Modbus] communications LED.

  • 4-pin CINCH Stick port on board allows firmware upgrades in the field for additional functionality without having to replace the hardware.
  • Power for the CeLAN RS422 Translator Module is provided by:
  • Rampart 12 VDC CeLAN power terminals.
  • CeLAN Remote Power Supply Module, or separate 12 VDC-power supply capable of delivering up to 25 mA.
  • Redundant CeLAN AES encrypted digital data bus technology provides secure, fast and reliable data transfer between the Rampart Panel and CeLAN modules.
  • Features:
  • Redundant AES Encrypted CeLAN communications.
  • RS422 Modbus communications.
  • Can be mounted in all compatible cabinet enclosures.
  • CINCH Stick field upgradeable software.

Specifications:

  • Power Requirements: 12VDC nominal, 10 VDC minimum, 14 VDC maximum, 40 mA minimum.
  • Panel Data Bus: CeLAN – 128 bit AES Encrypted.
  • Wire gauge supported:  18AWG – 22AWG.
  • Dimensions:  4.5” x 3.4” (114.3 mm x 86.4mm).
  • Operating Temperature: 32° F to 120° F  (0° C to 49° C), up to 140° F (60° C) under temporary conditions
  • Maximum Humidity: 85+/- 5% relative humidity, noncondensing, 86° F +/- 3° F.

Multimode Fiber Conversion Module

Ce-FC-R

 

The CeLAN Fiber Conversion Module expands the versatility of Protector and Protector II Panel by allowing the CeLAN to be converted from standard copper wires to a dual fiber. On Protector and Protector II Systems, you can connect multiple fiber conversion modules to a single panel as needed to provide secure communications.

 

The fiber optic communication performance can readily be determined via the on-board LED Fiber Strength Meter. The meter uses 5 LEDs  - one red and 4 green  to display the strength of the fiber signal coming in from the other CeLAN Fiber Conversion Module and is a readily recognized in a ‘cell phone’ like signal strength display.

Government Agencies, Sensitive Compartmented Information Facilities - SCIF’s, Command Centers. Critical Infrastructure, Embassies, High Security Critical Industrial, Energy Infrastructure, Transportation, Financial Institutions and any facility requiring the highest level of encrypted security.

Data Bus Isolator

Ce-Isolator

 

The CeLAN Isolator Module provides filtering of unwanted signals on the CeLAN data bus. Have confidence installing in a noisy environment.

 

Clean-up the “noise” from:

  • Lighting
  • Audio Components
  • Power Generators
  • High Voltage

When to use it:

  • Remove damaging A/C induced on the data line due to improper wiring with high voltage wiring or other noisy electronic wiring.
  • To remove any unintended intelligence bearing signals that may me riding on the data line.
  • Extend the length of the CeLAN. Protect systems from damaging ground loops.
  • Isolate one set of CeLAN devices from another set of CeLAN devices.
