CeLAN Door-EZ Door And Security Gate Control

Ce-Door-EZ

 

The Ce-Door-EZ Module is a microprocessor based standalone door controller system or a multi-door/gate system when used with a CINCH Control Panel. Complies with the new 2016 UL 325 requirement. The 2x16 display provides real time clear, easily understood instructions for operations, programming, maintenance and diagnostics. All system inputs and outputs are protected against transients and static discharges. All communication Interfaces are optically isolated.

Supports trouble free 4-Conductor data wiring for simplified, easy to install systems. Security hardening options include - Core UL 1076 security platform to build on for system integration needs. Communication lines to all modules utilize AES data packet encryption. 5.7” color touchscreen option with user and admin. login passwords option.

Features:

  • AES Encrypted CeLAN communications.
  • Supervised digital inputs. The supervision of digital inputs is via 3K ohm EOL resistor.
  • Seven Form A relay outputs, One form A.
  • 12 and 24 VDC outputs for control functions.
  • Single encrypted CeLAN data port.
  • Removable terminal strips ease of installation.
  • Multi-level override vs. older master/slave setup.
  • Supports HMI automation touchscreens, and manual buttons.
