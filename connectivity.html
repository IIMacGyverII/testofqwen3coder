CeLAN Printer Module

Ce-P

 

The CeLAN Printer Module allows you to connect a parallel printer to any panel or receiver. The module allows you to print (or log) events as they occur. The module also includes a zone input that can be used as a tamper or any other system input.

  • CeLAN AES encrypted technology for security, easy setup and enhanced performance.
  • Single 25-pin “Centronics” Parallel Connector.
  • One (1) fully supervised input zone.
  • Five (5) state zones:
  • Open Circuit
  • Supervised Circuit (3.0k EOL)
  • Short Circuit
  • Ground Fault High
  • Ground Fault Low
  • On-board LED power and CeLAN bus status indicators.
  • Includes Accessory Hardware kit.
  • CINCH Stick Field Upgradeable firmware.

CeLAN Remote Memory Module

Ce-RMM

 

The CeLAN Remote Memory Module (RMM) captures system events on a removable SD card that can be read on a PC. The module also includes a zone input that can be used as a tamper or any other system input.

 

Features:

  • Allows for up to 32 GB of system event history.
  • Redundant, Primary and Secondary SD slots allows for continuous recording of events while allowing review of recent events.
  • Single Input Zone for use as tamper, or any other zone application.
  • Mounts inside Protector and Protector II Accessory Enclosure Includes Accessory hardware kit.
  • CINCH Stick upgradeable firmware.

CeLAN Remote Power Module

Ce-RP

 

The CeLAN Remote Power Module is a dual-application device that allows use in either stand-alone mode or in fully supervised CeLAN expansion module mode.

 

Stand-Alone Mode:

  • Supervised 5 Amp, 12 VDC power supply.

Two on-board Form-C relays provide supervision status of:

  • AC input to the module
  • Back-up battery status
  • AES Encrypted CeLAN communications
  • Supervised 5 Amp 12 VDC Power Supply
  • One hardware input zone per module
  • Five (5) state zones:
  • Open Circuit
  • Supervised Circuit (3k EOL)
  • Short Circuit
  • Ground Fault High
  • Ground Fault Low
  • Two (2) Form-C relays that are fully programmable and controllable by the Protector and Protector II system.
  • On-board LED power and bus status indicators.
  • On-board LED current-load indication.
  • CeLAN AES encrypted technology for security, easy setup and enhanced performance.
  • ‘CINCH Stick field Upgradeable software.

CeLAN Expansion Module

CeEX

 

The CINCH Systems CeLAN Expander Module (EX), simplifies your installation by allowing you to interconnect up to 10 different CeLAN devices / wire-runs (+12 VDC, Bus A, Bus B, GND), on separate terminals. It also provides three sets of additional power and ground terminals for other powered devices. The module can be mounted into the Protector and Protector II Enclosure, Protector and Protector II Accessory Enclosure, or Custom Plastic CeLAN Enclosure w/wall & cover tamper.

 

Features:

  • Ten 4-position terminal blocks for CeLAN Expansion (+12 VDC, Bus A, Bus B, GND).
  • Three 2-position terminal blocks (+12 VDC, GND), for power expansion to other powered devices.
  • Mounts inside Protector and Protector II, Protector and Protector II Accessory, or Enclosure- CeLAN Custom Plastic w/wall & cover tamper.
  • Includes Accessory hardware kit.

Data Bus Ce-Isolator

Ce-Isolator

 

The CeLAN Isolator Module provides filtering of unwanted signals on the CeLAN data bus. Have confidence when installing in a “noisy environment”.

 

Clean-up the “noise” from:

  • Lighting
  • Audio Components
  • Power Generators
  • High Voltage

When to use it:

  • Remove damaging A/C induced on the data line due to improper wiring with high voltage wiring or other noisy electronic wiring.
  • To remove any unintended intelligence bearing signals that may me riding on the data line.
  • Extend the length of the CeLAN. Protect systems from damaging ground loops.
  • Isolate one set of CeLAN devices from another set of CeLAN devices.
