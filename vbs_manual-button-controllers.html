CeLAN Manual Barrier Controller Module

Ce-MBC

 

Expands the versatility of Rampart System by adding the capability to use manual buttons to also control the operation of vehicle barriers or other similar devices. On Rampart Systems, you can connect both touch screen controls (Ce-TS-VBS) along with manual controls (Ce-MBC) to operate the same or different barriers.

 

Power for the CeLAN Manual barrier Controller Module is provided by:

  • Rampart 12 VDC CeLAN power terminals.
  • CeLAN Remote Power Supply Module, or separate 12 VDC-power supply capable of delivering up to 25 mA.
  • CeLAN AES encrypted digital data bus technology provides secure, fast and reliable data transfer between the Rampart Panel and CeLAN modules.

Features:

  • AES Encrypted CeLAN communications.
  • Four (4) hardware input zones per module
  •  [ Key Switch EFO Reset, EFO push button, Up (Secure) push button, Down (Un-Secure) push button ]
  • Allows for “by barrier” EFO and reset option.
  • On-board LED power and CeLAN bus status indicators.
  • CeLAN AES encrypted technology for security, easy setup and enhanced performance.
  • Three (3) 6 VDC LED outputs per module.
  •  [ Up (Secure) LED, Down (Un-Secure) LED, EFO LED ].
  • Can be mounted in all compatible cabinet enclosures.
  • CINCH Stick field upgradeable software.

Specifications:

  • Power Requirements: 12VDC nominal, 10 VDC minimum, 14 VDC maximum, 40 mA maximum.
  • Panel Data Bus: CeLAN – 128 bit AES Encrypte.
  • Output current limitations (per LED output):5 mA @ 6 VDC.
  • Output type: 3x open collector rated.
  • Dimensions:  5.1” x 1.9”  (129.5 mm x 48.3mm).
  • Operating Temperature: 32° F to 120° F  (0° C to 49° C), up to 140° F (60° C) under temporary    conditions.
  • Maximum Humidity: 85+/- 5% relative humidity, noncondensing, 86° F +/- 3° F.

CeLAN Remote Controller Panel System Module

Ce-RCP-SB

 

The CINCH Systems CeLAN Remote Controller Panel System Module expands the versatility of Rampart System by adding the capability to use manual buttons to also control the operation of vehicle barriers, gates or other similar devices.

Connect both touch screen controls (Ce-TS-VBS) along with manual controls (Ce-RCP) to operate the same or different gates. The Ce-RCP acts as the main controller module for any manual button controller unit and does not actually operate any of the barriers or gates.  The Ce-RCP allows for the manual enabling or locking out of Ce-MBCs, Ce-MGCs, Ce-MMSs or remote Ce-RCPs. The Ce-RCP logically enables or disables other controls, it does not just simply interrupt the power to slave controllers. The logical enabling or disabling of the remote controller units protects against the attacker simply opening up the remote controller box and manually connecting power locally to operate a barrier.

Power for the CeLAN Remote Controller Panel System Module is provided by:

  • Rampart 12 VDC CeLAN power terminals.
  • CeLAN Remote Power Supply Module, or separate 12 VDC-power supply capable of delivering up to 25 mA..
  • CeLAN AES encrypted digital data bus technology provides secure, fast and reliable data transfer between the Rampart Panel and CeLAN modules.

 

Features:

  • AES Encrypted CeLAN communications.
  • Six (6) hardware input zones per module.
  • [System Power Key Switch –Off/On/Reset, Remote 1 Power Key Switch –Off/On,  Remote 2 Power
  • Key Switch –Off/On,  Annunciator –Off/On]
  • On-board LED power and CeLAN bus status indicators.
  • CeLAN AES encrypted technology for security, easy setup and enhanced performance.
  • Four (4) 6 VDC LED outputs per module.
  •  [Annunciator – Continuous, Annunciator – Pulsed, LED 1, LED 2].
  • Can be mounted in all compatible cabinet enclosures.
  • CINCH Stick field upgradeable software.

Specifications:

  • Power Requirements: 12VDC nominal, 10 VDC minimum, 14 VDC maximum, 40 mA minimum.
  • Panel Data Bus: CeLAN – 128 bit AES Encrypted.
  • Output current limitations (per LED output):5 mA @ 6 VDC.
  • Output type: 4x open collector rated.
  • Dimensions:  5.1” x 1.9”  (129.5 mm x 48.3mm).
